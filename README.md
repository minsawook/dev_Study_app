## 사용된 기술 및 패키지

- get : 효율적인 상태 관리를 강화하여 UI 구성 요소 처리 및 업데이트 프로세스를 단순화합니다.
- image_picker : 기기에서 이미지를 가져옵니다.
- image_cropper : 이미지를 자릅니다.
- auto_size_text : 글자크기를 크기에 맞게 자동으로 변경합니다.
- uuid : 랜덤 글자 생성
