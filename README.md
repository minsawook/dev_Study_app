# dev_studygroup_app

개발자들의 스터디를 지원하고, 프로젝트 팀 협업을 촉진하며, 활발한 커뮤니티를 육성하도록 설계된 플랫폼인 Dev Study Group 앱에 오신 것을 환영합니다.
플러터와 파이어 베이스를 사용하여 구성하였습니다.

# 스터디 그룹 만들기
 - 사용자 정의 가능한 이름과 설명으로 스터디 그룹을 만듭니다.
 - 팀원을 추가하고 직책을 할당합니다.
 - 스터디 그룹 회원 신청을 관리합니다(수락 또는 거부).

## 사용된 기술 및 패키지

- get : 효율적인 상태 관리를 강화하여 UI 구성 요소 처리 및 업데이트 프로세스를 단순화합니다.
- image_picker : 기기에서 이미지를 가져옵니다.
- image_cropper : 이미지를 자릅니다.
- auto_size_text : 글자크기를 크기에 맞게 자동으로 변경합니다.
- uuid : 랜덤 글자 생성
